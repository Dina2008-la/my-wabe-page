## âœï¸ Customization Guide

### Personal Information

1. **Update your name and bio** in `index.html`
   - Look for sections with `[Your Name]`, `[Your Location]`, etc.

2. **Add profile photo**
   - Replace `images/profile.jpg` with your photo
   - Recommended size: 500x500px minimum

3. **Update project details**
   - Edit project information in `pages/projects.html`
   - Add project images to `images/` folder
   - Update GitHub links and live demo links

4. **Modify About page**
   - Edit `pages/about.html` with your background and skills
   - Update education, experience, and qualifications

5. **Update Resume**
   - Edit `pages/resume.html` with your information
   - Add PDF resume to `files/` folder

6. **Social Links**
   - Replace placeholder URLs with your actual profiles
   - Update email address throughout the site

7. **Color Scheme**
   - Edit CSS variables in `css/styles.css` (top of file) to change colors:
     ```css
     :root {
       --primary-color: #2563eb;
       --secondary-color: #1e40af;
       --accent-color: #f59e0b;
       /* ... other colors ... */
     }
     ```
